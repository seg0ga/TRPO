#### Задание:

Необходимо создать интерфейсные тесты для тестирования части функционала библиотеки линейной алгебры OpenBLAS.

Описание работы

Скачать дистрибутив OpenBLAS (https://github.com/OpenMathLib/OpenBLAS)

Создать интерфейсные тесты, покрывающие весь CBLAS функционал вашего задания (только функции на С!).
на одной ОС;

* c одним или несколькими потоками (threads);

* для всех типов аргументов и значений.

Запустить интерфейсные тесты скомпилированные с библиотекой OpenBLAS:

* продемонстрировать успешное прохождение;
* продемонстрировать полноту покрытия.
